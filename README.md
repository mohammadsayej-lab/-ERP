# 📊 نظام إدارة الموارد البشرية والمحاسبة المتكامل (ERP System)

## 🎯 نظرة عامة
نظام متكامل مصمم لربط إدارة الموارد البشرية مع المحاسبة المالية، يوفر:
- ✅ إدارة الموظفين والرواتب
- ✅ المحاسبة المالية والقوائم المالية
- ✅ إدارة الفواتير والمشتريات
- ✅ نظام التقارير المتقدمة
- ✅ نظام الأمان والصلاحيات (RBAC)
- ✅ نظام المراجعة والتدقيق (Audit Trail)

## 🏗️ هيكل المشروع

```
-ERP/
├── src/
│   ├── WebAPI/                 # خادم الـ API الرئيسي
│   ├── Core/                   # طبقة العمل الأساسية
│   ├── Infrastructure/         # طبقة الوصول للبيانات
│   ├── Shared/                 # المشاركات والـ DTOs
│   └── Frontend/               # التطبيق الأمامي (اختياري)
├── Database/
│   ├── Scripts/                # سكريبتات SQL
│   └── Migrations/             # ترقيات قاعدة البيانات
├── Tests/                      # اختبارات الوحدة والتكامل
├── Docs/                       # التوثيق
├── docker-compose.yml          # تكوين Docker
└── README.md                   # هذا الملف

```

## 🚀 البدء السريع

### المتطلبات:
- .NET 7.0+
- SQL Server أو PostgreSQL
- Visual Studio 2022 أو VS Code

### الخطوات:
1. استنساخ المستودع:
```bash
git clone https://github.com/mohammadsayej-lab/-ERP.git
cd -ERP
```

2. تثبيت المتطلبات:
```bash
dotnet restore
```

3. تكوين قاعدة البيانات:
```bash
dotnet ef database update --project src/Infrastructure
```

4. تشغيل التطبيق:
```bash
dotnet run --project src/WebAPI
```

## 📋 المميزات الرئيسية

### 1. إدارة الموارد البشرية
- إدارة بيانات الموظفين
- حساب الرواتب والمستحقات
- إدارة الحضور والانصراف
- إدارة الإجازات والرخص

### 2. المحاسبة المالية
- إدارة دليل الحسابات
- تسجيل القيود اليومية
- التقارير المحاسبية
- الأرصدة والأرباح والخسائر

### 3. التكامل بين الأنظمة
- ربط تلقائي بين الرواتب والحسابات المالية
- توليد القيود المؤتمتة
- معالجة الاستقطاعات والضرائب
- حساب المستحقات نهاية الخدمة

### 4. الأمان والصلاحيات
- نظام تحكم بالوصول المستند على الأدوار (RBAC)
- تشفير البيانات الحساسة
- نظام المراجعة والتدقيق الشامل
- تسجيل جميع العمليات المالية

## 🔐 الأمان

الحساسات الأمنية المطبقة:
- Authentication: JWT Token
- Authorization: Role-Based Access Control
- Encryption: AES-256 للبيانات الحساسة
- Audit Trail: تسجيل جميع العمليات
- Input Validation: التحقق من جميع المدخلات
